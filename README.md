# Aaron Franceschi Portfolio

Welcome to my personal portfolio website! This site showcases my skills, projects, certifications, and experience as a developer. It's built with **Next.js**, **TypeScript**, and **Tailwind CSS**, featuring smooth scrolling, responsive design, and modern UI animations.

---

## 🚀 Features

- **Responsive Design** – Works seamlessly on desktop, tablet, and mobile devices.
- **Interactive Navbar** – Smooth scroll to sections with active link highlights.
- **Certifications & Badges** – Displayed in a polished, animated marquee.
- **Projects Showcase** – View portfolio projects with hover effects.
- **Skills Section** – Highlights technical skills with badges.
- **Contact Form** – Easy way to get in touch.
- **Modern UI/UX** – Dark theme with subtle gradients and animations.

---

## 🛠 Tech Stack

- **Next.js** (React framework)
- **TypeScript** for type-safe development
- **Tailwind CSS** for styling
- **Framer Motion** (for subtle animations)
- **react-scroll** (smooth scrolling)
- **react-fast-marquee** (for certification badges)
- **Vercel / Cloudflare Pages** (recommended for deployment)

## ⚡ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/aaronfranceschi/portfolio.git
cd portfolio
````

### 2. Install Dependencies

```bash
npm install
# or
yarn install
```

### 3. Run Development Server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view your portfolio locally.

---

## 📦 Build & Deploy

### 1. Build for Production

```bash
npm run build
# or
yarn build
```

### 2. Start Production Server

```bash
npm start
# or
yarn start
```

### 3. Deployment Recommendations

* **Vercel**: Fully compatible with Next.js; supports server-side rendering, static exports, and API routes.
* **Cloudflare Pages**: Works with Next.js static exports.
